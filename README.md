# Project Pemrograman Sistem
Project Lampu Otomatis &amp; Pencegah Kerumunan di Dalam Ruangan

### Alat & Bahan

- Arduino UNO 1pcs
- Breadboard 1pcs
- Relay 1pcs
- Sensor Ultrasonik 1pcs
- Lampu 1pcs
- Kabel 2meter
- Colokan 1pcs
- Fitting Lampu 1pcs
- Kabel Jumper 20pcs
- Resistor (220ohm) 1pcs
- Panel Lcd 1pcs
- Buzzer 1pcs
